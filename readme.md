1️=> What is the difference between var, let, and const?

var, let, and const are used to declare variables in JavaScript. var is function-scoped and can be redeclared and reassigned. let is block-scoped and can be reassigned but not redeclared, while const is block-scoped and cannot be reassigned or redeclared.

2️=> What is the spread operator (...)?

The spread operator (...) is used to expand elements of an array or object. It helps combine arrays, copy arrays, or add new properties to objects in a simple way.

3️=> What is the difference between map(), filter(), and forEach()?

map() creates a new array by transforming each element. filter() creates a new array with elements that match a condition. forEach() simply loops through the array and performs an action but does not return a new array.

4️=> What is an arrow function?

An arrow function is a shorter way to write functions in JavaScript using =>. It makes the code more concise and is commonly used in modern JavaScript, especially with array methods like map() and filter().

5️=> What are template literals?

Template literals are a way to create strings using backticks ( ). They allow variables and expressions to be inserted inside a string using ${} and support multi-line strings easily.
